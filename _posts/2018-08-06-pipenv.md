## Pipfile

### Example

```
[[source]]
url = "https://pypi.python.org/simple"
verify_ssl = true
name = "pypi"

[dev-packages]

[requires]
python_version = "3.6"

[packages]

flask = "==0.12.2"
requests = "==2.18.4"

```
