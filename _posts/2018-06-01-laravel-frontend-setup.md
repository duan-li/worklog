Install laravel
`composer create-project --prefer-dist laravel/laravel project`


Host laravel
`php artisan serve`



Install npm components
```
npm install
```

some time maybe 
```
npm install laravel-elixir --save-dev
```

add `/node_modules` to `.gitignore` file
