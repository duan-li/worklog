* `⌘`: Command key
* `⌥`: Option key (ALT in windows)
* `^`: Control key
* `⇧`: Shift key

Description | Shortcut
------------ | ------------- 
Search everywhere | Double `⇧` 
Recent file | `⌘` + `E` 
Duplicate line | `⌘` + `D`
Move line up | `⌘` + up 
Move line down | `⌘` + down
