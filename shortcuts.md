## Key sign

Windows | Mac OS
------------ | -------------
`Ctrl` | `^`
`Alt` | `Option` `⌥`

* `⌘`: Command key
* `⌥`: Option key (ALT in windows)
* `^`: Control key
* `⇧`: Shift key
